FocusHub

FocusHub es una aplicación desarrollada en Flutter diseñada para mejorar la productividad del usuario mediante una interfaz intuitiva, tareas organizadas y sesiones de enfoque tipo Pomodoro. Está pensada como una solución multiplataforma, incluyendo su versión web. Este documento describe en detalle el proceso de desarrollo, estructura, tecnologías, despliegue en GitHub Pages y buenas prácticas aplicadas.

Book&Go

Este proyecto tiene como finalidad desarrollar una plataforma que permita gestionar reservas en distintos ámbitos (como restaurantes, hoteles y centros culturales) desde un único lugar, mejorando la experiencia del usuario y facilitando la organización y estructura de los establecimientos.

Proyecto AWS con NodeJS

Este proyecto tiene como objetivo desplegar una aplicación sencilla desarrollada en Node.js en una máquina virtual de AWS (EC2), conectarla a una base de datos MySQL gestionada por RDS, servir archivos estáticos desde un bucket S3, y automatizar el despliegue continuo usando GitHub Actions.
